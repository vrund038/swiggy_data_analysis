# 🍽️ Swiggy Data Analysis Project

## 📌 Project Overview

This project focuses on analyzing Swiggy food delivery data to extract meaningful insights using **Python**.
The goal is to solve key business problems (KPIs) and visualize trends to understand customer behavior, restaurant performance, and order patterns.

---

## 🎯 Objectives

* Analyze Swiggy dataset to uncover insights
* Solve important **KPI-based business problems**
* Perform **data cleaning and preprocessing**
* Create **visualizations** for better understanding
* Build a complete **data analysis workflow**

---

## 🛠️ Tech Stack

* **Python**
* **Pandas** – Data manipulation
* **NumPy** – Numerical operations
* **Matplotlib / Seaborn** – Data visualization
* **Jupyter Notebook / VS Code**

---


## 📈 Visualizations

* 📊 Bar charts (Top restaurants, categories)
* 📉 Line charts (Order trends over time)
* 🥧 Pie charts (Category distribution)

---

## 🔍 Data Analysis Workflow

1. Data Collection
2. Data Cleaning (handling null values, duplicates)
3. Data Transformation
4. KPI Calculation
5. Data Visualization
6. Insights Generation


---

## 📌 Sample Insights

* Certain restaurants dominate order volume
* Discount pricing impacts sales significantly
* Specific food categories are more popular
* Peak ordering times can be identified

---

## 🧠 Learning Outcomes

* Hands-on experience with real-world data
* Improved data cleaning and preprocessing skills
* Strong understanding of KPI analysis
* Data visualization techniques
